# Companion UI — Vision

## 一句话
对话驱动的伴生式 Gen UI，让 AI agent 的表达从文字扩展到交互式视觉。

## 灵感
Her OS1 / Mercy Maddox / JARVIS / Swan Song — AI companion 的交互范式。

## 核心原则
1. Canvas 是舞台，对话是旁白
2. 视觉跟随对话自动伴生，用户不需要操作
3. 只在文字说不清的时候才可视化
4. 基于 agentic-lite 库，纯前端实现

## 技术选择
- agentic-lite 做 LLM 调用
- React + Vite 做前端
- Framer Motion 做动画
- 纯前端，无自建后端
